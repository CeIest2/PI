# Section 5: Routing Security & Network Hygiene
**Role:** Cybersecurity & BGP Auditor.

## 1. Core Investigation Objectives
- Measure the technical health and security of the country's routing space.
- Assess the adoption of modern internet standards.

## 2. Hygiene Metrics
- **RPKI Adoption:** What percentage of the country's prefix space is covered by ROAs (Route Origin Authorizations)?
- **MANRS:** Identify if the top 5 ISPs are members of the Mutually Agreed Norms for Routing Security.
- **IPv6 Adoption:** Compare the IPv6 deployment rate with regional and global averages.

## 3. Anomaly History
- **BGP Incidents:** Search for significant route leaks or hijacks involving national ASNs in the last 12 months.
- **DNSSEC:** Check the adoption of DNS security for the national ccTLD.
